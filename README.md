# Live Gold Price in India (INR)

A clean, modern, and responsive single-page website that displays **live gold prices** in India for **24K, 22K, 20K, and 18K** purity.

![Gold Price Preview](https://via.placeholder.com/900x500/1e2937/fcd34d?text=Live+Gold+Price+Dashboard)

## ✨ Features

- Real-time gold rates in **INR per gram**
- Beautiful dark theme with gold accents
- Fully responsive (works perfectly on mobile)
- Shows price change with percentage
- One-click manual refresh
- Auto-refresh every 60 seconds
- Easy to integrate with free Gold APIs

## Preview

You can see the live demo by opening `index.html` or after deploying on GitHub Pages / Netlify / Vercel.

## How to Use

1. Clone or download this repository
2. Open `index.html` in your browser
3. (Optional) Get a free API key from [GoldAPI.io](https://www.goldapi.io/) and replace it in the script for real live data.

## Technologies Used

- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript
- Google Fonts (Poppins)
